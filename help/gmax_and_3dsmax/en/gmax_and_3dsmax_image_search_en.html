Image Search

You can help 3ds max (or gmax) to find images by specifying folders for searching them. To do it, go to the main menu, and select item "Customize->Configure Paths" (or "Customize->Configure User Paths"). A dialog with tabs will be shown. Then go to tab "Bitmaps" (or "Bitmaps/Photometrics", or "External Files"). You will see a window like this:

Click the "Add..." button. A new window will appear:

Find the root texture folder, check the "Add Subpaths" checkbox, and press the "Use path" button. If that's all right, now you can see a list of all folders which will be used to find images.

Click the "OK" button to apply changes and close this window.
